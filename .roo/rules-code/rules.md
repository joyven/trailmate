# 全局约定

- 人机交互使用中文，也就是你与我的对话使用中文。
- 回答问题时，每次回答前增加称呼：”主人“，这用于检测规则是否生效。
- 生成的页面如果没有特殊强调，页面内容是日文。

# 项目规则

- 项目使用 vue3+vite 构建，使用 typescript 增强 js 代码类型。
- css 使用 Tailwind.
- 网站是响应式的，支持 PC 端和手机端。
-

# 项目构成

- 网站原型是 figma 生成，vscode 中我们已经安装配置了`Framelink MCP for Figma`MCP 服务。
- 网站导航有景点介绍（スポット紹介）、特产美食（グルメ·特産ペ-ジ）、体验打卡（グルメ·特産ペ-ジ）、仓敷魅力（倉敷の魅力を伝える特集ペ-ジ）、交通与常见问题（アクセス・FAQ）。
- 在景点介绍的页面底部，有推荐路线，引入推荐路线（おすすめコース）页面。
